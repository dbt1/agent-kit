# memory/

Host-aware project memory generated by bootstrap.

Current layout:

- `memory/projects/<project-id>/shared.md`
- `memory/projects/<project-id>/hosts/<host-id>.md`
- `memory/projects/<project-id>/index/<host-id>.md`

Rules:

- `shared.md` is for host-independent knowledge.
- `hosts/<host-id>.md` is for host-local paths, tools, quirks, and validation.
- `index/<host-id>.md` is generated and should not be edited directly.

Legacy `*_MEMORY.md` files may still exist during migration.
Generated project memory remains ignored by default to avoid noisy git status.
